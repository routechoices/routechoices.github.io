Routechoices - GPS Tracking Solution - on Github

Routechoices is a GPS tracking solution available throught its website at routechoices.com.

It is open source and its code is all available on Github github.com/routechoices

Repositories

The code is split in different repositories

Routechoices Server

Repo

The code for the server.

It is a Python/Django project.

Routechoices iOS Tracker App

Repo

The code for the iOS app that people can download to share their live GPS position.

It is a Swift project.

Routechoices Android Tracker App

Repo

The code for the android app that people can download to share their live GPS position.

It is a Kotlin project.

Trackers Config

Repo

Contains default configurations file for routechoices compatibles GPS tracking devices.

Mapant Tile Proxy

Repo

Code for a proxy that convert tiles from custom projections to WGS84 standard projection.

It is a Python/Flask project.

NGINX Config

Repo

Contains configurations for NGINX as setup on the routechoices.com server.
